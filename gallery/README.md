# 📸 پوشه گالری

اینجا برای قرار دادن عکس‌های خود است!

## چطور استفاده کنم؟

1. **عکس‌های خود را اینجا قرار دهید**
   - فایل‌های تصویری (jpg, png, webp) را اینجا قرار دهید
   
2. **فایل را سمت درست نامگذاری کنید**
   - مثال: `piano-practice.jpg` یا `concert-photo.png`

3. **فایل `src/data/gallery-photos.js` را ویرایش کنید**
   - برای هر عکس یک خط اضافه کنید:
   ```javascript
   photo(
     "/gallery/نام-عکس.jpg",
     "عنوان عکس",
     "توضیح کوتاه",
     "1403/06/09"
   ),
   ```
